# Life Counseling

人生の辛さに寄り添う対話パートナー AI Agent

## 概要

このプロジェクトは、Claude Code を使った対話型カウンセリングツールです。専門家として診断や治療を行うのではなく、相談者が自分自身の苦しみを理解し、それと共に生きる力を見出す手助けをします。

## 使い方

```bash
cd life-counseling
claude
```

対話を始めると、AI が傾聴と問いかけを中心に対話を進めます。

## コマンド

| コマンド | 説明 |
|---------|------|
| `/counseling:save` | 今日の対話をスナップショットとして保存する |
| `/counseling:review` | 過去の対話を振り返り、気づきを読み返す |
| `/counseling:values` | 自分にとって大切なことを記録・振り返る |
| `/counseling:continue` | 前回の対話の続きから始める |
