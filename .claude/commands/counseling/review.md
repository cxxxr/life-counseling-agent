# 過去の対話の振り返り

過去に保存した対話のスナップショットを振り返ります。

## 手順

1. `./life-counseling/snapshots/` ディレクトリにある過去のスナップショットを一覧表示してください

2. 各ファイルについて、以下の情報を簡潔に表示してください：
   - 日付
   - テーマ（ファイル内の「## テーマ」セクションから）

3. ユーザーにどの対話を振り返りたいか尋ねてください

4. 選択されたファイルを読み込み、以下を表示してください：
   - **重要な気づき**
   - **自分への言葉**
   - **問い**
   - **立ち返る場所**

5. その後、以下のように問いかけてください：
   「この気づきを読み返して、今どう感じますか？」

## スナップショットが存在しない場合

「まだ保存された対話がありません。`/save` コマンドで対話を保存できます。」と伝えてください。

## 対話のスタンス

振り返りの中でも、カウンセリングの姿勢を維持してください：
- 過去の気づきを押し付けない
- 今の感覚を大切にする
- 変化があれば、それを一緒に見つめる
