# 価値観の記録と振り返り

自分にとって大切なこと（価値観）を記録し、振り返ります。

## 保存先
`./life-counseling/values.md`

## 手順

### 1. 既存の価値観ファイルを確認

`values.md` が存在する場合は、現在記録されている価値観を表示してください。

### 2. ユーザーに問いかける

以下のいずれかを尋ねてください：

- 「今日、新しく気づいた大切なことはありますか？」
- 「これまでの価値観を振り返って、何か変化はありますか？」
- 「今の生活は、ここに書かれている価値観に沿っていると感じますか？」

### 3. 価値観の追加・更新

ユーザーが新しい価値観を共有した場合、以下の形式で追記してください：

```markdown
## YYYY-MM-DD に気づいたこと

（価値観の内容）

---
```

### ファイルが存在しない場合

新規作成し、以下のヘッダーを追加してください：

```markdown
# 私にとって大切なこと

このファイルには、対話の中で見つけた「自分にとって大切なこと」を記録しています。

---
```

その後、以下のように問いかけてください：

「あなたにとって、本当に大切なことは何ですか？
すぐに答えが出なくても大丈夫です。
最近、心が動いた瞬間や、『これは大事だ』と感じたことがあれば、教えてください。」

## 対話のスタンス

- 価値観は変わりうるものとして扱う
- 「正しい価値観」を押し付けない
- 「認められたい」と「感動で震える」のように、複数の価値観が共存することを認める
- 価値観と現実の間にギャップがあっても、それを責めない
